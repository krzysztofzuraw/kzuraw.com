# kzuraw.com

This repository contains code & blog posts for my [webpage](https://kzuraw.com).

## How to run

Install dependencies:

```shell
pnpm install
```

Run development server:

```shell
pnpm dev
```

Run production build:

```shell
pnpm build
```

Generate an empty blog post with required frontmatter

```shell
pnpm post-create
```
